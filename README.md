# YG GeoPortal V5.1

## Upload ke GitHub
1. Ekstrak ZIP.
2. Buka folder `YG-GeoPortal-V5.1`.
3. Pilih SEMUA isi folder: index.html, webgis.html, css, js, assets, data.
4. Drag semuanya ke halaman `uploading an existing file` GitHub.
5. Commit changes.
6. Settings → Pages → Deploy from a branch → main → /(root) → Save.
